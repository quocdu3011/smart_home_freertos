# FreeRTOS Basic Test

Sketch: `FreeRTOS_Basic_Test.ino`

## How to use

1. Install the library that provides `Arduino_FreeRTOS.h`
2. Upload this sketch
3. Open Serial Monitor at `115200`
4. Check for:
   - `FreeRTOS test booted.`
   - repeated `FreeRTOS heartbeat`

## Interpretation

- If this works, your board and FreeRTOS setup are okay
- If this does not compile or does not print, fix the FreeRTOS library first
